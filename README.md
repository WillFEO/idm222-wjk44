# idm222-wjk44

Via Ferrata Informational Website

This website is my project for IDM222, Winter Quarter 2025-2026. I have created a site about via ferratas, a unique kind of hike that I really enjoy. The site includes a description of what via ferratas are, what gear is needed, what difficulty levels exist, where they can be found, and a personal anecdote about my experience with them.

Tech Used: HTML, CSS, GitHub, VSCode, Figma, Vercel
